# Features
<ul>
  <li> Signup and signin </li>
  <li> Login permanence with local storage</li>
  <li> Create and delete habits </li>
  <li> List habits </li>
  <li> Look for today's habits </li>
  <li> Mark habit as done </li>
</ul>
